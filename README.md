# Emotion Recognition from Speech

Deep Learning based Speech Emotion Recognition system using CNN and Speech Signal Processing.

---

# Features

- Emotion detection from speech audio
- MFCC feature extraction
- CNN Deep Learning model
- Streamlit Web App
- Real-time audio prediction
- Confusion matrix visualization
- Training curve visualization

---

# Dataset

RAVDESS Dataset

Emotions:
- Neutral
- Calm
- Happy
- Sad
- Angry
- Fearful
- Disgust
- Surprised

---

# Tech Stack

- Python
- TensorFlow
- Keras
- Librosa
- Streamlit
- Scikit-learn
